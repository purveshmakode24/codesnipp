# Insert Data in mongoDB

> Simple example in which you can insert (users) data in monogoDB using Mongoose ODM and Express framework.

### Installation
- Install dependencies
```
npm install
```

### Running
```
npm start
```
- Head over to localhost:3000 in your browser.
